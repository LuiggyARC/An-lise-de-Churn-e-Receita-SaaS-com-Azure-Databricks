# An-lise-de-Churn-e-Receita-SaaS-com-Azure-Databricks
Este projeto tem como objetivo construir um pipeline analítico em Azure Databricks para analisar churn e receita de uma empresa SaaS fictícia. A arquitetura segue o padrão Lakehouse com camadas Bronze, Silver e Gold, permitindo transformar dados brutos em métricas de negócio acionáveis.
